# OpSta Works — Demo

**Hotel Operations Intelligence Platform**

A browser-based internal portal for hotel operations staff — built to streamline training, guest services, and shift communications across multi-property hospitality environments.

---

## 🏨 Demo Pages

| Page | Description |
|------|-------------|
| `index.html` | Login page with role-based demo accounts |
| `dashboard.html` | Main operations dashboard with live MOD communications |
| `room-reference.html` | Searchable room directory with filters |
| `concierge.html` | Local attractions & guest recommendations |
| `amenities.html` | Property amenities, hours & rates |
| `training.html` | Staff training hub with progress tracking |

## 🚀 Features

- **MOD Communications** — Real-time shift messaging and handoff log
- **Room Reference** — Full directory with type, floor, view, ADA, and connecting room filters
- **Concierge Guide** — Categorized local attractions with distance and tips
- **Training Hub** — Onboarding modules with progress tracking
- **Amenities Guide** — Hours, rates, and guest-facing information
- **Role-Based Access** — Admin, Manager, and Staff tiers
